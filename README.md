## Installation

### npm

```sh
npm i react-css-spinners
```

### yarn

```sh
yarn add react-css-spinners
```

### unpkg

```html
<script src="https://unpkg.com/react-css-spinners@latest/dist/bundle.umd.js"></script>
```

## Usage

1. Import any spinner

```js
import { Ellipsis } from 'react-css-spinners'
```

2. Use it as usual

```js
<Ellipsis />
```
